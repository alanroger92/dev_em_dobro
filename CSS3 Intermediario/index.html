Especifcidade Efeito Cascata

O efeito cascata é um dos princípios fundamentais do CSS, onde as regras de estilo são aplicadas em uma ordem específica. A especificidade é um mecanismo que determina qual regra de estilo deve ser aplicada quando há conflitos entre várias regras que se aplicam ao mesmo elemento.

A especificidade é calculada com base em quatro categorias de seletores: seletores de ID, seletores de classe, seletores de tipo e seletores universais. A regra com a maior especificidade é aplicada ao elemento. Se duas regras tiverem a mesma especificidade, a última regra definida no código CSS será aplicada.

Por exemplo, um seletor de ID tem uma especificidade mais alta do que um seletor de classe, e um seletor de classe tem uma especificidade mais alta do que um seletor de tipo. Portanto, se houver uma regra de estilo para um elemento com um seletor de ID e outra regra para o mesmo elemento com um seletor de classe, a regra com o seletor de ID será aplicada.

Estilos de Texto

Os estilos de texto em CSS3 permitem que você controle a aparência do texto em uma página web. Você pode usar propriedades como font-family, font-size, color, text-align, entre outras, para personalizar o estilo do texto.

Além disso, CSS3 introduziu novas propriedades para estilizar o texto, como text-shadow, que permite adicionar sombras ao texto, e text-decoration, que pode ser usado para criar sublinhados, linhas sobrepostas ou linhas através do texto.
