Posicionamento: Static, Relative e Absolute

O posicionamento em CSS é uma parte fundamental para controlar a disposição dos elementos em uma página web. Existem três tipos principais de posicionamento: static, relative e absolute. Cada um deles tem suas próprias características e usos específicos.

1. Static

O posicionamento static é o valor padrão para todos os elementos. Os elementos com posicionamento static são posicionados de acordo com o fluxo normal do documento. Eles não respondem a propriedades como top, right, bottom ou left.

2. Relative

O posicionamento relative permite que um elemento seja posicionado em relação à sua posição normal. Ele pode ser movido usando as propriedades top, right, bottom e left, mas o espaço original do elemento ainda é reservado no layout.

3. Absolute

O posicionamento absolute remove o elemento do fluxo normal do documento. Ele é posicionado em relação ao seu elemento pai mais próximo que tenha um posicionamento diferente de static. Se não houver um elemento pai com posicionamento, ele será posicionado em relação ao documento inteiro.
