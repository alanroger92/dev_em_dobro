Tags Semânticas

As tags semânticas são elementos HTML que descrevem claramente o significado do conteúdo que elas envolvem. Elas ajudam a melhorar a acessibilidade, a SEO (Search Engine Optimization) e a estrutura do documento. Aqui estão algumas das principais tags semânticas:

  • <header>: Representa o cabeçalho de um documento ou seção.
  • <nav>: Define um conjunto de links de navegação.
  • <main>: Indica o conteúdo principal do documento.
  • <article>: Representa um conteúdo independente e auto-contido, como um artigo de blog ou notícia.
  • <section>: Define uma seção genérica de um documento, usada para agrupar conteúdo relacionado.
  • <aside>: Representa conteúdo tangencialmente relacionado ao conteúdo principal, como uma barra lateral ou caixa de informações.
  • <footer>: Define o rodapé de um documento ou seção, geralmente contendo informações de contato, direitos autorais ou links relacionados.
