Para que serve class e id?

Class e ID são atributos HTML usados para identificar e estilizar elementos específicos em uma página web. Atributo "class" é usado para agrupar elementos que compartilham estilos ou comportamentos semelhantes, permitindo aplicar regras CSS ou scripts JavaScript a todos os elementos com essa classe. Já o atributo "id" é único para cada elemento na página, servindo como um identificador exclusivo. Ele é frequentemente usado para aplicar estilos ou scripts a um único elemento específico. Em resumo, "class" é para grupos de elementos, enquanto "id" é para elementos individuais.
